My name is Rusty Shackleford. I'm a hobbyist programmer from the UK. My favourite languages are Ruby and C - I also like Perl, D and Rust to a lesser extent.

My main OS is OS X at the moment. It may seem like an unpopular opinion, but it feels like Fancy Unix to me. When I'm not using OS X, I'm using Linux. More specifically, Arch Linux. I like how bare it is and ABS. I use Windows too - for a HTPC, running XBMC and Steam. That's about it. My favourite editor is obviously Vim and MacVim. I also use Brackets for web development and XCode for larger projects.

Being a hobbyist, I prefer working on my own, small projects.
Occationally I'll fork a project, but generally only develop stuff I need. It's nice if someone else finds a use for them, or likes them though. Sometimes I even make websites.

All my stuff is open-source, and licensed under either 3-clause BSD, MIT or the very comical WTFPL.
