My name is George Watson. I'm a hobbyist programmer from the UK. My favourite languages are C, Ruby and Python. I have experience in JS / Coffescript, Shell, C++ and Perl - and have experimented with Objective-C, PHP, D and C#.

The main OS I work on is OS X at the moment. It may seem like an unpopular opinion, but it feels like it Just works. When I'm not using OS X, I prefer using Linux (Debian). Out of all the distros I've tried, Debian is probably my favourite. I've used Windows all my life but I don't like it much. I really just use it for playing Dark Souls and for a front-end for Plex. That's it really. My editor of choice is Vim. Sometimes I'll use Xcode (with Vim bindings, of course).

Being a hobbyist, I prefer working on my own, small projects - so any spaghetti code you see is just because I do it on purpose (I like doing it). Occationally I'll fork a project, but generally only develop my own stuff. It's nice if someone else finds a use for them, or likes them though. I also make games. I enjoy engine work more than actual game logic. Sometimes I even do web-design, but not so often. I'm not great at design.

All my stuff is open-source, and licensed under either 3-clause BSD, MIT or the very comical WTFPL.
